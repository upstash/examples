## Getting Started

1. Create an Upstash database and get the connection url and token.
2. Add them to your `.env` file
```env .env
UPSTASH_REDIS_REST_URL=
UPSTASH_REDIS_REST_TOKEN=
```

3. Run the development server:

```bash
npm run dev
# or
yarn dev
```
